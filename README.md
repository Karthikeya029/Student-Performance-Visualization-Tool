# 🎓 EduTrack – Student Performance Management System

## 📌 Overview
EduTrack is a full-stack web application to manage student marks, attendance, and performance using real-time updates and analytics.

## 🚀 Features
- Role-based dashboards (Student, Teacher, Coordinator)
- JWT authentication
- Real-time updates (Socket.IO)
- Data visualization (Chart.js)

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB + MySQL

## ▶️ How to Run

1. Install dependencies:
   npm install
2. Create `.env` file using `.env.example`
3. Start server:
   npm start
4. Open:
   frontend/login.html
